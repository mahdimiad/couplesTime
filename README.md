# couplesTime

An iOS app designed for couples to share expenses and manage tasks together.

## Overview

couplesTime helps couples stay organized by providing a shared platform to:
- Track and split expenses
- Manage shared tasks and to-dos
- Stay connected on financial and household responsibilities

## Features

- Expense sharing and tracking
- Task management
- Built with SwiftUI for a modern iOS experience

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Getting Started

1. Clone the repository
2. Open `couplesTime.xcodeproj` in Xcode
3. Build and run the project
