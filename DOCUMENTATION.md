# iOS Development Documentation & References

This file contains documentation links and references for the couplesTime iOS app project.

## App Description

**couplesTime** is an iOS app designed for couples to manage shared expenses and tasks together. The app helps couples stay organized, reduce daily stress, and build stronger relationships through better communication and partnership.

### Core Purpose
Stop fighting, start thriving. couplesTime eliminates common relationship frustrations like "Did you pick up milk?" or "Who paid for what?" by providing a simple, shared platform for tracking tasks and expenses.

### Key Features

#### The Anti-Nag System
- Tasks with love, not demands
- Gentle reminders and appreciation when things get done
- Smart timing for sensitive requests
- Instant appreciation with hearts and thank you notes

#### Fair Share Made Simple
- Visual contribution balance
- Automatic expense splitting
- Chore equity tracking
- See your contributions at a glance

#### Smart Couple Intelligence
- AI that learns your patterns and suggests task assignments
- Location-based reminders for both partners
- Smart shopping list merging
- Auto-suggests who should handle what

#### Life Planning Together
- Shared calendar with smart scheduling
- Date night planning with budget
- Goal setting and milestone celebration
- Plan everything as a team

#### Money Without Drama
- Real-time expense tracking
- Custom split percentages
- Spending insights and trends
- Track shared expenses transparently

#### Always in Sync
- Instant sync across phones and tablets
- Offline mode for when you're apart
- Push notifications done right
- Real-time updates across all devices
